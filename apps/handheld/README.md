# Handheld operations application

The handheld client supports QR-based inventory entry and exit, relocation, requests, reporting, order gathering, and production assignment. Its layouts and controls are optimized for touch screens and one-handed industrial workflows.

Run it from the repository root with `npm run dev:handheld`. Camera access requires HTTPS except on `localhost`; configure the API through `apps/handheld/.env` using the supplied example.
